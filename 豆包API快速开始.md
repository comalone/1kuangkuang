# 豆包API快速开始指南

## 🚀 5分钟快速上手

### 步骤1: 获取豆包API密钥

1. 访问 [火山引擎控制台](https://console.volcengine.com/)
2. 注册/登录账号
3. 进入"豆包大模型"服务
4. 创建API密钥
5. 复制您的API Key

### 步骤2: 配置API密钥

**方法A: 使用配置助手(最简单)**

双击运行 `配置豆包API.bat`,按提示输入API密钥即可。

**方法B: 手动配置**

打开命令行,执行:
```bash
setx DOUBAO_API_KEY "your-api-key-here"
```

### 步骤3: 启动服务

双击运行 `一键启动.bat`

看到以下信息表示启动成功:
```
[INFO] 所有线程已启动,系统就绪!
```

### 步骤4: 使用系统

1. 浏览器访问 `http://localhost:8000`
2. 对着摄像头伸出食指
3. 保持静止1秒设定起点
4. 移动食指框选区域
5. 再次静止1秒完成截图
6. 在结果页面右上角切换到"豆包"
7. 查看AI分析结果

## 🎯 API切换说明

### 在结果页面切换API

结果页面右上角有三个元素:
- **状态显示**: 显示当前使用的API
- **通义千问按钮**: 切换到阿里云API
- **豆包按钮**: 切换到豆包API

点击按钮即可切换,未配置的API会显示为灰色。

### 切换效果

- ✅ 立即生效,无需重启
- ✅ 绿色提示显示切换成功
- ✅ 下次截图使用新的API

## 📊 两个API的区别

### 通义千问 (Qwen-VL)
- ✅ 阿里云官方模型
- ✅ 中文理解优秀
- ✅ 响应速度快
- ⚠️ 需要上传到OSS

### 豆包 (Doubao Vision)
- ✅ 字节跳动官方模型
- ✅ 视觉理解强大
- ✅ 直接base64传输
- ⚠️ 大图片体积较大

## 🔧 常见问题

### Q1: 配置后提示"未配置API Key"?

**解决方法:**
1. 确认环境变量已设置: `echo %DOUBAO_API_KEY%`
2. 重新打开命令行窗口
3. 重新运行 `一键启动.bat`

### Q2: 切换API后没反应?

**解决方法:**
1. 检查浏览器控制台是否有错误
2. 确认API密钥正确
3. 检查网络连接

### Q3: 分析速度慢?

**解决方法:**
1. 检查网络连接
2. 尝试切换到另一个API
3. 减小截图区域

### Q4: 如何测试API是否正常?

**解决方法:**
运行测试脚本:
```bash
python test_doubao_api.py
```

## 💡 使用技巧

### 技巧1: 选择合适的API

- **解题场景**: 推荐通义千问(数学理解好)
- **图片描述**: 推荐豆包(视觉理解强)
- **文字识别**: 两者都很好

### 技巧2: 优化截图质量

- 保持手部稳定
- 确保光线充足
- 框选区域不要太小
- 避免模糊和反光

### 技巧3: 自定义提示词

编辑 `prompts.json` 添加自定义提示词:
```json
{
  "name": "我的提示词",
  "content": "请详细分析这张图片..."
}
```
